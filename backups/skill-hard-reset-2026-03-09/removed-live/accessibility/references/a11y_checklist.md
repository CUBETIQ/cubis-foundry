# Accessibility checklist (OneUp HR)

## Labels & semantics
- [ ] Every button has a clear label (or Semantics label)
- [ ] Icon-only buttons have Tooltip + Semantics
- [ ] Images have alt/semantic labels if meaningful

## Text scaling
- [ ] No overflow at large text sizes
- [ ] Avoid fixed heights for text containers
- [ ] Use flexible layouts

## Focus & keyboard (desktop)
- [ ] Tab order is logical
- [ ] Focus visible
- [ ] Dialogs trap focus appropriately
- [ ] Escape closes dialogs if expected

## Color & contrast
- [ ] Don’t rely on color alone for status
- [ ] Contrast acceptable for text on backgrounds
