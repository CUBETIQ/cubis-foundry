# Foundry Design Datasets

This directory stores Foundry-owned normalized design datasets.

- Inputs may come from external design research such as Impeccable, Anthropic-aligned skill lineage, and selected public UX references.
- Agents should consume these normalized datasets rather than scraping or copying external pages directly.
- Every entry preserves source and license metadata so design guidance does not become anonymous prompt residue.

Current datasets:

- `style-directions.json`
- `layout-patterns.json`
- `mobile-patterns.json`
- `component-motifs.json`
- `token-language.json`
- `motion-patterns.json`
