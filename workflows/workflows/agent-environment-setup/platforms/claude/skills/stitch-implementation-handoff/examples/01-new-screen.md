# Example: New Screen From Stitch

Prompt shape:

> Use the finalized Stitch billing screen artifact and implement it in my Next.js app with the existing design system.

Expected workflow:

1. Confirm the final Stitch artifact is current.
2. Identify the destination route, layout shell, and shared components in the repo.
3. Implement the screen with project tokens and primitives instead of raw generated markup.
4. Summarize any intentional deviations from Stitch.
