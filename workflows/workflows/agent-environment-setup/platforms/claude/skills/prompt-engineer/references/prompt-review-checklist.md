# Prompt Review Checklist

Load this when reviewing a prompt, system message, skill description, or agent rule.

## Review Points

- Is the task objective concrete?
- Is the output format explicit?
- Are prohibited actions or boundaries stated clearly?
- Are user-provided or external instructions isolated from trusted instructions?
- Does the wording overfit to one example instead of defining the rule?
- Is there enough context to trigger the right skill without bloating the prompt?
