# Example: Crash Triage

Prompt:

> Reproduce the crash on the notifications settings screen, clear logs before the repro, and give me a concise bug report with screenshot evidence, the failing action, and the relevant log lines.

Expected behavior:

1. Clear logs.
2. Navigate to the failing screen.
3. Reproduce once.
4. Capture screenshot, current activity, and filtered logs.
5. Report root symptom, likely trigger, and smallest next debugging step.
