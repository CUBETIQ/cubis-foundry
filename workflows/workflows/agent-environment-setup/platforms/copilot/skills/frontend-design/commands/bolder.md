# /bolder

Increase visual impact: stronger typography, bolder colors, more contrast.

## What It Does

Dials up the visual intensity of the design. Makes headlines more commanding, colors more saturated, spacing more dramatic, and overall presence more confident. Use when a design feels timid or forgettable.

## Boldness Techniques

### Typography

- Increase headline weight to 700–900
- Increase headline size by 1–2 steps on the type scale
- Add more contrast between heading and body sizes (ratio ≥ 1.5x)
- Consider a display or condensed font for primary headings

### Color

- Increase chroma/saturation on accent colors
- Darken text colors for stronger readability
- Add bold background sections to break visual monotony
- Use color blocking — full-width colored bands

### Layout

- Increase padding and margins — generous whitespace amplifies boldness
- Use larger images and media
- Make CTAs bigger and more prominent
- Add dramatic scale contrast between elements

### Contrast

- Push light/dark contrast further
- Use reversed sections (dark background, light text) for emphasis
- Increase border weight on key elements

## Usage

- `/bolder` — make the entire design more impactful
- `/bolder typography` — focus on type hierarchy
- `/bolder hero` — make the hero section commanding
