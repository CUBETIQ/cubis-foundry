# /critique

UX design review focused on hierarchy, clarity, emotional resonance, and user flow.

## What It Does

Evaluates the design from a user's perspective. Assesses whether the interface communicates clearly, guides attention effectively, and creates the right emotional response.

## Review Areas

1. **Visual hierarchy** — Is there a clear reading order? Does the most important content stand out?
2. **Clarity** — Can a first-time user understand what to do within 5 seconds?
3. **Emotional resonance** — Does the design create the intended feeling? (trust, excitement, calm, urgency)
4. **User flow** — Is the path from entry to goal obvious and frictionless?
5. **Differentiation** — Does it look distinctive or generic? Would a user remember it?
6. **Consistency** — Are patterns consistent throughout? Same action = same look?

## Output Format

Rate each area (Strong / Adequate / Needs Work), then provide 3-5 specific, actionable improvements prioritized by impact.

## Usage

- `/critique` — full UX review
- `/critique hero` — critique just the hero section
- `/critique checkout flow` — critique the checkout user flow
