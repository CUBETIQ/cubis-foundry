# /quieter

Reduce visual noise: softer colors, lighter weights, more whitespace.

## What It Does

Dials down the visual intensity. Makes the design calmer, more refined, and easier on the eyes. Reduces competing elements so content can speak. Use when a design feels loud, cluttered, or overwhelming.

## Quieting Techniques

### Typography

- Reduce heading weight to 500–600 (medium/semibold)
- Decrease size contrast between headings and body
- Increase line-height for more breathing room
- Use a lighter text color for secondary content

### Color

- Reduce chroma/saturation across the palette
- Use tinted grays instead of pure gray
- Limit accent color usage to one or two key elements
- Soften borders from solid to subtle tints

### Layout

- Increase whitespace between sections
- Reduce the number of visual elements per viewport
- Remove decorative elements that don't serve a purpose
- Use subtle dividers (thin lines, spacing) instead of heavy borders

### Visual Weight

- Replace filled icons with outlined variants
- Use lighter box-shadows or remove them entirely
- Reduce image saturation or apply subtle overlays
- Soften corner radii (sharp corners feel louder)

## Usage

- `/quieter` — make the entire design calmer
- `/quieter colors` — desaturate and soften the palette
- `/quieter layout` — add breathing room and reduce density
