# /distill

Simplify the UI: remove clutter, reduce cognitive load, clarify hierarchy.

## What It Does

Identifies unnecessary complexity in the interface and removes it. Focuses on content hierarchy, reducing visual noise, and ensuring every element earns its place on screen.

## Simplification Principles

1. **Remove before restyle** — if an element doesn't serve a clear purpose, remove it
2. **Reduce visual weight** — lighter borders, fewer colors, quieter backgrounds
3. **Merge related elements** — combine redundant labels, consolidate actions
4. **Strengthen hierarchy** — make the primary action obvious at a glance
5. **Increase whitespace** — give remaining elements room to breathe

## What to Look For

- Redundant labels or headings
- Too many competing calls-to-action
- Visual elements that don't communicate information
- Overly complex layouts that could be flattened
- Decorative elements that add noise without meaning

## Usage

- `/distill` — simplify the entire view
- `/distill navigation` — simplify navigation structure
- `/distill form` — reduce form complexity
