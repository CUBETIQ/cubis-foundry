# Example: Dashboard Prompt Enhancement

Raw request:

> Make our dashboard feel cleaner and more premium with Stitch.

Improved prompt:

```markdown
A calm, premium analytics dashboard for finance operators with restrained typography, strong hierarchy, and dark-on-light contrast.

Platform: web, desktop-first
Theme: light, editorial, precise, quietly premium
Design system:
- Primary action uses a deep navy accent with crisp monochrome surfaces
- Typography favors sharp hierarchy and compact data labels
- Cards have restrained radius and low-noise elevation

Structure:
1. Overview header with page title, date filter, and export action
2. KPI grid with four high-contrast metric cards
3. Revenue trend chart beside top accounts table
4. Recent alerts panel with severity badges

Edit scope:
- Improve hierarchy, spacing rhythm, and action emphasis
- Preserve the information architecture and data density
```
