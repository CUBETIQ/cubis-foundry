# Implement Track Workflow

Use this workflow for larger work that benefits from milestone tracking.

## Steps
1. Split implementation into clear milestones.
2. Complete one milestone at a time.
3. Validate each milestone before continuing.
4. Publish concise progress and remaining risk.
