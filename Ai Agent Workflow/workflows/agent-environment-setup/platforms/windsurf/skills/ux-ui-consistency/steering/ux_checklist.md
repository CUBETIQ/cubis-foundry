# UX checklist (OneUp HR)

## Screen structure
- [ ] Clear title + primary action (if any)
- [ ] Content uses OneLayout / OneCard / OneListTile patterns consistently
- [ ] No visual jitter when loading transitions

## States
- [ ] Loading state present
- [ ] Error state with retry present
- [ ] Empty state with guidance/action present
- [ ] Data state present

## Lists
- [ ] Pull-to-refresh present where appropriate
- [ ] Pagination has bottom loader and no duplicate requests
- [ ] Stable keys on items if expansion/selection/stateful rows

## Forms
- [ ] Field validation and error display is consistent
- [ ] Submit disabled while submitting
- [ ] Success feedback (snackbar/dialog) consistent

## Feedback & micro-interactions
- [ ] Snackbars used sparingly and consistently
- [ ] Navigation transitions feel consistent
